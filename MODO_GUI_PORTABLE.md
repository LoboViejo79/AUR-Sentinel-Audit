# AUR Sentinel Audit - GUI Portable

## Ejecutar

```bash
chmod +x autorun.sh
./autorun.sh
```

## Reportes

El programa genera:

- HTML
- PDF
- TXT
- JSON

Al presionar **Generar reporte HTML**, podrás abrir el HTML, abrir el PDF o abrir la carpeta de reportes.

## Modo portable

Las dependencias Python se instalan dentro de `.venv/`, no en el sistema.
